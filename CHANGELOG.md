# Changelog

## 0.1.0 (unreleased)

Initial release. A tool that helps you talk to people better.

### Features

- 24-question communication style assessment (DISC-based, own questions)
- 8-question quick-tag wizard for typing other people
- 80 handcrafted coaching cards across 5 situations (feedback, request, conflict, pitch, difficult news)
- Coach-first flow: value in under 2 minutes, assessment is opt-in depth
- People management with DISC wheel visualization and confidence indicators
- Journal with 1-5 outcome tracking and optional notes
- Adaptation insights with per-type trends, friction detection, and growth tips
- Shareable insight cards via public URLs
- Offline-capable PWA with IndexedDB persistence
- Dark mode (auto/light/dark with system preference)
- Data export as JSON
- Mobile-first design with 44px touch targets and safe-area support
